# 🛸 Rick and Morty — Website Example

Site desenvolvido em **março/abril de 2026** pelos desenvolvedores **Thiago Rodrigues dos Santos** e **Isabella Sakamoto**.

O projeto foi criado durante o **1º semestre do curso de Desenvolvimento de Sistemas Multiplataforma** na FATEC de Ilha Solteira, utilizando tecnologias modernas de front-end para construção de interfaces interativas e performáticas.

---

## 🚀 Sobre o Projeto

Explore o multiverso de **Rick and Morty**!

O site permite que o usuário:

* 🎬 Assistir ao trailer da **9ª temporada**
* 📺 Navegar por todas as temporadas da série
* 🔗 Acessar as temporadas disponíveis na plataforma **MAX**

O foco do projeto foi criar uma experiência moderna, rápida e imersiva para o usuário.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GSAP (animações e interatividade)

---

## ✅ Boas Práticas Aplicadas

* Estrutura semântica em HTML
* SEO otimizado
* Imagens otimizadas (WEBP e JPG)
* Organização modular de arquivos
* Código limpo e sem redundâncias
* Performance e carregamento otimizado

## 📁 Estrutura do Projeto

```
/
│── index.html
│── 1temporada.html
│── 2temporada.html
│── ...
│── 8temporada.html
│── favicon.png
│
├── assets
│   ├── css
│   │   ├── main-min.css      # Estilos globais e reset
│   │   ├── style.css         # Estilização da página inicial
│   │   └── temporadas.css    # Estilo das páginas de temporadas
│   │
│   └── js
│       ├── script.js         # Scripts da página inicial
│       └── temporadas.js     # Scripts das temporadas
│
└── media
    ├── icons                 # Ícones utilizados no site
    ├── images
    │   ├── webp              # Imagens otimizadas WEBP
    │   └── jpg               # Imagens JPG
    ├── logos                 # Logos utilizadas
    ├── thumbs                # Thumbnails dos episódios
    └── videos                # Vídeos do preloader
```
